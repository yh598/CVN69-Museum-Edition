# Release Notes — v0.1.0 Hull

This immutable milestone release provides the complete hull-only production set for the USS Dwight D. Eisenhower Museum Edition.

Highlights: parameterized 476 mm full hull, three hidden-key modules, engraved waterline, anchor pockets, four socketed shaft lines, eight A-bracket struts, four five-blade propellers with blind bores, twin keyed rudders, allowed-palette multi-material 3MF, and complete manufacturing documentation.

Validation: PASS, 34/34 checks. Bambu Studio reports the 21-part 3MF manifold. See `QA/VALIDATION.md` and `QA/validation_report.json`.
