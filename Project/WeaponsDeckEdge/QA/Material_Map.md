# Milestone 4 Object Material Map

| Object family | Filament | Rationale |
|---|---|---|
| Sponsons, foundations, launcher/CIWS bodies, boat, cradle, davit, railings, ladders, lockers | Bambu PLA Matte Ash Gray | primary exterior structure |
| RAM and Sea Sparrow launcher faces; boat insert | Bambu PLA Matte Charcoal | dark openings/recesses with no paint |
| CIWS barrels and selected metallic external detail | Bambu PLA Silk Silver | selected metallic detail |
| CIWS radar domes and life-raft canisters | Bambu PLA Matte Ivory White | visible light-colored housings/canisters in public imagery |
| Navigation-light housings | Bambu PLA Basic Black | dark external housings; lens colors not asserted |

Materials are assigned by object name in every named-object 3MF. No AMS slot number is fixed, and no filament outside the user's existing palette is required.
